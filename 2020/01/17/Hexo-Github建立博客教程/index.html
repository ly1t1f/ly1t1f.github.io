前言

一、前言

为什么要建立博客，如果使用果平台博客如博客园等，就会发现有一个自己的博客，就会极为方便，不受约束，没有广告，布局也可以有着自己的想法。所以有自己的博客是一件很酷的事。

Hexo+GitHub

二、Hexo+GitHub

为什么选择Hexo+GitHub，因为成本较低，只有学习成本;
GitHub_Pages会提供免费空间，并会提供username.github.io的默认域名，则不需要购买服务器和域名。

搭建环境

三、搭建环境

1、系统：ubuntu18.04
2、安装node.js

在终端下输入：sudo apt-get install nodejs，安装完成即可

3、安装git

终端输入sudo apt-get install git，安装完成即可

4、安装Hexo

使用npm安装Hexo

终端下输入：
npm install -g hexo-cli

配置GitHub

四、配置GitHub

1、注册登录GitHub

创建新的仓库
,右上角选择Create a new repository，命名为username.github.io
[外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传(img-PPPofdua-1579064711158)(https://pic1.zhimg.com/80/v2-d7ffab98b05bbb00ce9602c6f8d0ce70_hd.jpg “仓库”)]

2、添加仓库密匙

在终端下输入：
ssh-keygen -t rsa -C "Github的注册邮箱地址"
一直enter即可完成，密匙生成完成后， 会得到 id_rsa和id_rsa.pub两个文件，打开id_rsa.pub文件并复制其内容；
打开密匙设置链接

将刚刚复制的内容复制到key对话框中，title可随意命名，点击Add SSH Key配置完成

五、初始化博客

1、建立本地存储文档

建立一个新的文件夹Blog，在本文件夹中打开终端，输入：
